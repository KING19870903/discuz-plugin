# baidu-searchbox/discuz-miniapp-plugin

小程序discuz插件，方便站长一键话生成app

## 快速开始

1. 首先要安装composer，参考<http://composer.baidu.com>
2. 将composer路径设置到$PATH环境变量
3. 执行`sh build.sh`即可

代码文件说明:

* `build.sh`: 构建脚本
* `ci.yml`: 集群构建的配置
* `composer.json`: Composer配置示例
* `src`: 源码目录
